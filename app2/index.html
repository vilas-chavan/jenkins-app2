Version 1.3 (V1.3) of Docker host

##Steps to implement AWS Pipeline##

1. Build Docker Image and Pushed on ECR
2. Create Dockerfile, Builsspec & defination file and pushed on codecommit
3. Create ALB, Targetgroup
4. Create hosted zone in Route53 and points to ALB
5. Create cluster, Task Defination and service
6. Create Project Build and Pipeline
7. Push the new changes on codecommit and observer the Pipeline
8. Access project www.skcloud.live
